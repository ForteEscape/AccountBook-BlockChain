## AccountBook-BlockChain
- WAP 2018년도 겨울방학 프로젝트

## 개발목표
- 블록체인을 활용한 장부시스템의 구현
- MYSQL 및 Python flask를 이용한 웹서비스 구현
- 기본적인 알고리즘과 실력 향샹을 목표
